# 🧮 Calculator Pro

Ein moderner Taschenrechner für den Browser mit einem übersichtlichen Design und schneller Bedienung.

## ✨ Funktionen

- ➕ Addition
- ➖ Subtraktion
- ✖️ Multiplikation
- ➗ Division
- 📐 Klammern
- 🔢 Kommazahlen
- ⌫ Löschen-Taste
- 📱 Für PC, Tablet und Smartphone geeignet
- 🎨 Modernes Design

## 🚀 Verwendung

1. Lade die Dateien herunter.
2. Öffne `index.html` im Browser.
3. Nutze den Taschenrechner direkt.

## 🛠️ Erstellt mit

- HTML
- CSS
- JavaScript

## 📄 Lizenz

Dieses Projekt kann frei genutzt, verändert und weiterentwickelt werden.
